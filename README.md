##  Prerequisites

- A C++ compiler (e.g. g++, clang, or MSVC)
- CMake (version 3.10+)

##  Build Instructions (Windows)

```bash
git clone https://github.com/LakhyarajRajkhowa/ClothSimulator.git
cd ClothSimulator
mkdir build
cd build
cmake ..
cmake --build .
